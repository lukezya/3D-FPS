------------------------------------------------------------------------------
CONTROLS
------------------------------------------------------------------------------
W - Run forward
A - Turn left and start running in that direction
D - Turn right and start running in that direction
S - Turn around and start running in that direction

1 - Change camera to first person camera
2 - Change camera to orbiting camera
3 - Change camera to third person camera

Left Shift - Walking, stop running

Spacebar - Jump

Mouse Left Click - Shoot gun

Change to First Person Camera to aim properly. The aim point is a tiny bit
higher than the centre of the aim

P - Open up camera settings menu to change distance and height of third person
camera and orbiting camera

Game spawns 100 random objects randomly with random materials but number of
objects spawned can be changed within Unity Project RandomizeObject c# Script.
